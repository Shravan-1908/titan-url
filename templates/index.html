Welcome to Titan URL

A modern URL shortener - shorten those long URLs!

Why use Titan URL?

FOSS

Titan URL is completely free and open source. The source code can be found on GitHub.

No account needed

You dont need to setup an account to use Titan URL. Just open the webpage and shorten the URLs.

Custom and random aliases

You can set the shortened URL aliases to custom as well as random ones. (Coming soon)

Secure

All the shortened URLs are stored in an encrypted form.
